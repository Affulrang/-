Диваны и кресла

Десктопное приложение

Позволяет импортировать данные из Excel-файла, с которыми можно будет работать: проводить фильтрацию, сортировку с возможностями создания/удаления новых данных и сохранения их в таблицу. Приложение позволяет пользоваться нужными функциями без сложностей в интерфейсе для более быстрой работы.

Мобильное приложение

Позволяет импортировать определенный Excel-файл и взять его текущие данные, с которыми можно будет работать: проводить фильтрацию, сортировку с возможностями создания/удаления новых данных и сохранения их в таблицу. Мобильное приложение создано с целью доступности работы вне компьютера.
